# Kinetic-GAN  
kinetic gan을 이용한 human action generative model을 학습하고 나아가 수어 데이터 생성을 수행하기 위한 repository 입니다.  

### KETI Dataset 적용  
KETI 데이터셋 적용을 위해 graph 코드를 작성하였습니다: graph_keti.py  
해당 코드는 GCN 연산을 위한 그래프 구성을 구하는 코드 입니다.
